# SKIMMED
### Solana MEV Personal Damage Report
**Colosseum Frontier Hackathon 2026**

> *"How much have bots stolen from you?"*

SKIMMED is a personal MEV intelligence tool for Solana users. Paste your wallet address — we pull your full swap history via **Dune SIM**, detect every sandwich attack with on-chain block verification, calculate your exact dollar losses using live token prices, and generate a forensic damage report narrated by **Claude AI**.

## The Problem

Sandwich bots quietly extract value from Solana users on every unprotected swap. Between $370M–$500M was extracted from ordinary users over a 16-month period. Most users have no idea it's happening, no way to see their historical losses, and no personalized guidance on how to stop it.

SKIMMED makes the invisible visible.

## Verification Tiers

SKIMMED doesn't just flag high slippage — it proves the attack happened on-chain.

| Badge | Meaning |
|---|---|
| `✓ confirmed` | Front-run tx + back-run tx found in same block. Bot wallet identified. |
| `~ suspected` | Excess slippage detected but block pattern inconclusive |
| `? unverified` | Block fetch unavailable — flagged by slippage heuristic only |

This makes SKIMMED's detection defensible to technical scrutiny. A suspected sandwich is never reported as confirmed.

## Dune SIM Integration

SKIMMED uses two Dune SIM endpoints:

- `GET /v1/svm/transactions/{address}` — full Solana swap history
- `GET /v1/svm/balances/{address}` — current wallet token balances

The **SIM Integration Strip** on every report shows the exact endpoint called and the 4-step flow from wallet input to MEV detection. The **Dev Panel** below it shows the raw JSON response with MEV-flagged fields highlighted in red — making the integration immediately clear to judges and users alike.

## How It Works

```
User pastes wallet
       ↓
Dune SIM fetches transaction history (/v1/svm/transactions)
       ↓
DEX swaps filtered (Jupiter, Raydium, Orca programs)
       ↓
Slippage delta calculated per swap
       ↓
Swaps > 1.2% excess slippage flagged as potential sandwich
       ↓
Helius getBlock fetches full block for each flagged swap
       ↓
Block scanned for front-run tx (before victim) + back-run tx (after victim)
       ↓
Same bot wallet in both = CONFIRMED sandwich
       ↓
Dollar losses calculated using live Jupiter prices
       ↓
Claude AI generates briefing + protection plan
       ↓
Report rendered with attack table, bot leaderboard, portfolio snapshot
```
